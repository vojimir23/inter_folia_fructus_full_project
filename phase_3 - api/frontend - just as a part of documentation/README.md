# Toy Example of the IFF Website

This is a toy example of the IFF website, featuring all filters, graphs, and text search functionality (made by Voja).

To use it, update the link in `script.js` at the beginning of the file so that it points to your active local or online endpoint.

By default, it is set to:

`const API_BASE_URL = "http://127.0.0.1:8000";`

> **Note:** The website may still contain bugs, but it is useful for demonstrating how easy it is to track requests sent to the API server.
